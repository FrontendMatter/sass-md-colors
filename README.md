# sass-md-colors

> The [Material Design Color Palette](https://www.google.com/design/spec/style/color.html#color-color-palette) as Sass variables.

## Install

```bash
npm install sass-md-colors
```

## Usage

```sass
@import "node_modules/sass-md-colors/colors/variables"

a {
	color: $red-500;
}
```
